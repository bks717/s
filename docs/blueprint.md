# **App Name**: LoomSheet

## Core Features:

- Data Entry Form: Form with fields for 'Primary Column', 'Sub-Columns for Width', 'Operator Name', 'Roll No', 'Width', 'Number 1', 'Number 2', 'Gr/Sut', 'Color', 'Lam/Unlam', 'Mtrs', 'G.W.', 'C.W.', 'N.W.', 'Average Loom No.', 'Production Date', and 'Variance'.
- Form Submission: Submit button to finalize and save data entered via the input form.
- Admin Authentication: Password-protected admin section to ensure secure access.
- Data Summarization: AI tool to generate summaries, comparisons, and/or flags based on Loom data fields. User will specify exactly what type of summary or report is to be created from the available data.
- Admin Dashboard: Dashboard displaying submitted data in a clear, readable format, organized in columns and rows for easy understanding and sorting.

## Style Guidelines:

- Primary color: Slate blue (#558BBD) for a calm and professional feel.
- Background color: Light gray (#F0F4F7) for a clean, neutral backdrop.
- Accent color: Warm orange (#D47A6A) to draw attention to key actions.
- Body and headline font: 'Inter' sans-serif for modern, clean readability.
- Form fields arranged logically for ease of data input, with clear labels.
- Admin dashboard presents data in a tabular format, sortable by column headers.
